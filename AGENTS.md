# Convenções do projeto

- Use PHP PSR-12, Form Requests, Policies e Services quando reduzirem responsabilidade.
- Todo código Vue deve usar TypeScript. Não crie arquivos JavaScript para aplicação ou configuração.
- Prefira páginas e componentes pequenos; extraia composables apenas para lógica reutilizável.
- Preserve os módulos opcionais desacoplados e remova os que não forem necessários ao iniciar um produto.
- Reviews devem priorizar bugs, segurança e regressões.
- Ao concluir uma funcionalidade, atualize testes automatizados e `docs/MANUAL_TESTS.md` quando aplicável.
